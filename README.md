# Alla Samba Siva Rao - Portfolio Website

🚀 **Live Site:** [View Portfolio](https://samba425.github.io/portfolio)

Modern, responsive portfolio website showcasing professional experience, projects, and technical skills.

## ✨ Features

- 🎨 Modern sky blue & green design with glass morphism
- 📱 Fully responsive (mobile, tablet, desktop)
- 🚀 High performance with smooth animations
- 🎯 Admin panel for easy content management
- 💼 Professional sections with detailed information

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript (ES6+)
- Font Awesome 6.4.0
- Google Fonts (Poppins)
- CSS Grid & Flexbox
- LocalStorage for admin panel

## 📂 Files

- `index.html` - Main portfolio page
- `admin.html` - Content management panel
- `styles.css` - All styling
- `script.js` - Animations and interactions
- `data.json` - Portfolio data

## 🚀 Quick Start

```bash
# Clone and open
git clone https://github.com/samba425/portfolio.git
cd portfolio
python3 -m http.server 8000
# Visit http://localhost:8000
```
